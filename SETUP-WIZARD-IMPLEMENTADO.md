## 🚀 **COMO USAR**

### **Instalação Automática**
```bash
# 1. Baixar projeto
git clone https://github.com/seu-usuario/pdv-fiodegala.git
cd pdv-fiodegala

# 2. Executar instalador
./install.sh

# 3. Configurar banco de dados
# Execute os scripts SQL no Supabase

# 4. Iniciar sistema
./start-all.sh

# 5. Acessar Setup Wizard
# http://localhost:5173
```


   ```bash
   chmod +x install.sh

   ```
